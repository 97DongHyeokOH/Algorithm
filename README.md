## 👨‍💻 Manual
1. 매주 2~4문제
2. 매주 수요일 🕘`21:00` 코드 리뷰
3. 각 주차별 Directory Name : `Week[num]`
    - num : 2자리 숫자로
    - ex) Week01, Week02
4. File Name : `문제번호_성씨`
    - 백준 : 1006_kang.cpp  
    - 프로그래머스 : 문제명_kang.cpp
    - 코드트리 : 문제명_kang.cpp
5. Commit Message : `[Solve] 💯 사이트명_문제번호.문제명`  
    - [Solve] 💯 BOJ_1006.습격자초라기
    - [Solve] 💯 programmers_level2.기능개발  
    - [Solve] 💯 codetree.십자모양폭발
